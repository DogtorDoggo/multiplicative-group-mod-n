### What is Multiplicative group mod $n$?

Given a positive integer $n$, the elements within $\{1, 2, \cdots, n - 1\}$ (bracket not showing up) that are coprime to $n$ forms a group, namely, multiplicative group mod $n$, noted as $\mathbb{Z}_n^\ast$, or $(\mathbb{Z}/n\mathbb{Z})^\times$ . By definition, every element $k \in \mathbb{Z}_n^*$ satisfies $\text{gcd}(k, n) = 1$.

Note: for $n > 1$, $\mathbb{Z}_n^\ast$ is non-empty since $1 \in \mathbb{Z}_n^\ast$.
